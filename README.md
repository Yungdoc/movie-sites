# Movie Site

This repository contains a static movie site built with plain HTML/CSS/JS.

Netlify setup:
- Publish directory: root (`.`)
- Build command: none (static site)

To deploy via Netlify UI: Connect your GitHub repo and select branch `main`.

To deploy via Netlify CLI:
1. Install: `npm i -g netlify-cli`
2. Login: `netlify login`
3. Link/create: `netlify init` (follow prompts)
4. Deploy: `netlify deploy --prod --dir=.`
